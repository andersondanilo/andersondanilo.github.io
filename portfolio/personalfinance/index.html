Portfólio: Personal Finance

Aplicativo para celular desenvolvido para o FirefoxOS, controle de contas a pagar e a receber voltado para uso pessoal.

Disponível em: https://marketplace.firefox.com/app/personal-finance

Repositório: https://github.com/andersondanilo/personalfinance

Participação

Desenvolvimento de todo o sistema, desde a idealização até a programação e tradução para Inglês e Espanhol.

Ferramentas Utilizadas

  • Javascript, CoffeScript
  • Css, Sass
  • Backbone, Zepto e Underscore
  • IndexedDB
  • Mocha (Teste Unitário)
