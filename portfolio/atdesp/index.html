Portfólio: ATDESP

Projeto desenvolvido para a Associação dos Trabalhadores Desempregados de São Paulo (ATDESP). Ofereçe a opção de cadastro de associados e gerenciamento dos mesmos. Possui Área do Associado e Painel para Administração.

Disponível em: http://atdesp.org.br

Participação

Desenvolvimento do cadastro de associados, gestão de documentos, contratos, cobranças e boletos.

Ferramentas Utilizadas

  • HTML, CSS, JavaScript
  • PHP 5.2
  • Yii Framework
  • MySQL
  • jQuery
